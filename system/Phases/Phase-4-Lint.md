# Phase 4: 檢查與維護 (Lint & Maintain)

確保維基系統的一致性與持續成長。

## 核心功能

### 一致性掃描
- 檢查資料不一致之處
- 標記格式錯誤
- 驗證連結完整性

### 資訊填補
- 識別缺失資訊
- 透過網路搜尋填補
- 自動更新相關文章

### 連接發現
- 找出概念間的新連接
- 識別可擴展的主題
- 建議新文章方向

### 探索建議
- 提出可進一步研究的問題
- 追蹤知識缺口
- 引導下一輪攝取

## 工作流程

```
維基 --> Lint 掃描 --> 報告 --> 修訂/填補 --> 返回 Phase 2
                 │
                 └--> 新連接建議 --> 新文章候選
```

## 操作步驟

1. **執行掃描**
   - 對 `wiki/` 執行全面檢查
   - 產出 `wiki/lint/` 報告

2. **處理報告**
   - 修正一致性问题
   - 填補缺失資訊
   - 建立新連接

3. **循環回 Phase 2**
   - 根據建議增強維基
   - 持續迭代

## 產出目錄

| 目錄 | 內容 |
|------|------|
| `wiki/lint/` | 檢查報告 |
| `wiki/lint/issues/` | 發現的問題 |
| `wiki/lint/suggestions/` | 改進建議 |

## 反饋循環

```
Phase 4 --> Phase 2 --> Phase 3 --> Phase 4 --> ...
              ↑                           │
              └───────────────────────────┘
```

維基系統透過此循環持續成長與進化。
